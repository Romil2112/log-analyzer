# Security Policy

## Supported versions

This is a free, open-source **demonstration / trial project**, not a commercial product
with a service-level agreement. Security fixes are made on a **best-effort** basis against
the latest `main` branch only.

| Version | Supported          |
| ------- | ------------------ |
| `main`  | ✅ best-effort      |
| older   | ❌                 |

## Reporting a vulnerability

Please report suspected vulnerabilities **privately** — do not open a public issue for a
security problem.

- **Email:** shahromil71321@gmail.com
- Include a description, affected component, and steps to reproduce if possible.

### What to expect

- **Acknowledgement** within **72 hours**.
- For confirmed issues, a fix or mitigation is targeted within **14 days** where feasible.
- Please practice **responsible disclosure**: give a reasonable opportunity to remediate
  before any public disclosure.

This project is provided "as is", without warranty, and is not backed by a commercial
support or SLA commitment.
